Store API Project Summary

Advance Filtering & Sorting with Mongoose and Express

Date: October 25, 2024

Overview

This project is a Store API built with Express and Mongoose. It allows for advanced filtering and sorting of products based on various criteria.

Major Learnings

  • Populating data from JSON and Using Mongoose,working with Postman for API testing
  • Using libraries for status code management
  • Extracting parameters from req.query
  • Creating dynamic query objects for Mongoose
  • Implementing advanced filtering and sorting
  • Handling pagination in API responses

Functionality

  • Dynamic Filtering: Allows filtering products based on fields like featured, company, and name using the GET /products endpoint.
  • Numeric Filtering: Supports various operators for numeric fields via numericFilter on fields like price, rating of the query.
  • Sorting: Sort results based on specified fields using the sort parameter.
  • Pagination: Implement pagination with limit and page query parameters.

Database Connection

The project connects to a MongoDB database using Mongoose to handle data management and querying.

Models

The Product model is defined using Mongoose schema in products.js.
